# EndlessDisk

Монтирование облачного хранилища VK Cloud S3 как обычного диска Windows с GUI-менеджером.

## Возможности

- Облачное хранилище как обычный диск в Проводнике Windows
- VFS-кэширование (настраиваемый размер, по умолчанию 20 ГБ)
- Контекстное меню (ПКМ) для файлов:
  - Копировать публичную ссылку
  - Открыть / закрыть доступ к файлу
- GUI-менеджер для всех настроек
- Автозапуск при старте Windows (без окон)
- Автоматическая установка rclone и WinFsp
- Полное удаление со всеми компонентами

## Требования

- Windows 10/11 (64-bit)
- PowerShell 5.1+ (предустановлен)
- Аккаунт VK Cloud с Object Storage (S3)

## Быстрая установка

### 1. Скачайте EndlessDisk

Скачайте или склонируйте репозиторий в любую папку, например `C:\EndlessDisk\`:

```
git clone https://github.com/CABi-Code/EndlessDisk.git C:\EndlessDisk
```

### 2. Запустите менеджер

```powershell
powershell -ExecutionPolicy Bypass -File "C:\EndlessDisk\VKDiskMenu.ps1"
```

Или дважды щёлкните по `VKDiskMenu.ps1` (если разрешено выполнение скриптов).

### 3. В окне менеджера

1. **Установите rclone** — нажмите кнопку «Установить» напротив rclone. Будет скачан автоматически.
2. **Установите WinFsp** — нажмите «Установить» напротив WinFsp. Потребуются права администратора.
3. **Введите ключи S3** — нажмите на блок «Ключи S3», введите Access Key и Secret Key из панели VK Cloud.
4. **Настройте параметры** — буква диска, бакет, домен, регион и другие.
5. **Нажмите «Сохранить»**.
6. **Подключите диск** — нажмите «Подключить диск».

### 4. Дополнительно

- **Контекстное меню** — установите через кнопку в разделе «Статус компонентов»
- **Автозапуск** — поставьте галочку «Запускать диск при старте Windows»
- **Ярлык** — поставьте галочку «Ярлык на рабочем столе»

## Получение ключей VK Cloud

1. Зарегистрируйтесь на [cloud.vk.com](https://cloud.vk.com)
2. Перейдите в **Объектное хранилище** → **Аккаунты**
3. Создайте аккаунт S3 и скопируйте **Access Key** и **Secret Key**
4. Создайте бакет (например, `vk-disk`)

## Настройки

Все настройки хранятся в `%APPDATA%\EndlessDisk\config.json`:

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| DriveLetter | V: | Буква диска |
| RcloneRemote | VKDisk | Имя remote в rclone |
| Bucket | vk-disk | Название бакета S3 |
| Domain | storage.cabi.world | Домен для публичных ссылок |
| EndpointHost | hb.bizmrg.com | Эндпоинт VK Cloud S3 |
| Region | ru-msk | Регион |
| CacheSize | 20G | Размер VFS-кэша |
| Transfers | 16 | Число параллельных потоков |

## Режимы запуска

```powershell
# GUI-менеджер (по умолчанию)
powershell -ExecutionPolicy Bypass -File "VKDiskMenu.ps1"
powershell -ExecutionPolicy Bypass -File "VKDiskMenu.ps1" gui

# Тихое монтирование (для автозапуска)
powershell -ExecutionPolicy Bypass -File "VKDiskMenu.ps1" mount

# Контекстное меню (вызывается автоматически из ПКМ)
powershell -ExecutionPolicy Bypass -File "VKDiskMenu.ps1" copylink "C:\path\file.txt"
powershell -ExecutionPolicy Bypass -File "VKDiskMenu.ps1" toggleacl "C:\path\file.txt"
```

## Удаление

Через GUI: нажмите «Полное удаление» в менеджере.

Будет предложено:
- Удалить весь конфиг rclone или только секцию EndlessDisk
- Удалить rclone.exe
- Удалить WinFsp

Удаляются: реестр (контекстное меню, автозапуск), ярлык, VBS-лаунчер, конфигурация приложения.

## Почему VK Cloud?

- Оплата по факту: ~1 руб/мес за 1 ГБ хранения
- Привязка карты и автооплата
- Быстрая регистрация с бонусными кредитами
- Полная совместимость с S3 API
